# OECD AI Governance and Classification

## What this skill does
Comprehensive workflow skill for classifying ai systems with the oecd framework, assessing governance needs across the ai lifecycle, and operationalizing the oecd ai principles.

## When to use it
Use this skill when you need a practical starting point for oecd ai governance and classification work and want the reusable workflow instructions in `SKILL.md` rather than a one-off note or draft.
