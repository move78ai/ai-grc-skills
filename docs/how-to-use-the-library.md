# How To Use The Library

## Choose an entry point
- Start with `umbrella/ai-grc-and-regulatory/SKILL.md` when the task is broad or you need routing help.
- Start with a standalone skill when the workflow family is already clear.

## Work pattern
1. Read the skill `README.md` for quick orientation.
2. Open `SKILL.md` for the actual operating instructions.
3. Use `references/`, `assets/`, and `scripts/` only when the skill points to them.

## Packaging rule
Source folders stay in the repo. Public release zip files are created later and attached to releases.
